

Install
=======

From the command line enter (in some cases you have to preced the
command with ``sudo``):


.. code-block:: bash

    $ pip install isbnlib

or:

.. code-block:: bash

    $ easy_install isbnlib

or:

.. code-block:: bash

    $ pip install isbnlib-3.5.4.tar.gz

(first you have to download the file!)
